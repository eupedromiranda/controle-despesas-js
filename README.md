# controle-despesas-js

Projeto simples de controle de despesas feito em HTML, CSS e JavaScript com visual similar aos visuais de app de banco

## Funcionalidades

- Adicionar despesas com nome, valor, categoria e nota
- Editar despesas já cadastradas
- Deletar despesas
- Filtrar despesas por categoria
- Ver o total de despesas
- Dashboard com gráfico de barras por categoria
- Modo escuro
- Layout responsivo, tipo app de banco

## Tecnologias

- HTML, CSS e JavaScript
- Chart.js para gráficos
